# 💳 Payment Processing System

Консольное Java-приложение для обработки денежных переводов между пользователями с акцентом на обработку исключений.

---

## 🚀 Функциональность

* 👤 Создание пользователей
* 💰 Пополнение баланса
* 💸 Переводы между пользователями
* 📊 История транзакций
* ⚠️ Обработка ошибок через кастомные исключения

---

## 🧠 Технологии

* Java Core
* OOP (инкапсуляция, слои)
* Collections (List)
* Optional
* Enum
* Exception Handling (кастомные исключения)

---

## 🧱 Архитектура

* `entity` — модели данных
* `repository` — хранение данных
* `service` — бизнес-логика
* `exception` — кастомные ошибки
* `ui` — консольный интерфейс

---

## ⚠️ Обработка ошибок

В проекте реализованы:

* DuplicateUserException
* UserNotFoundException
* InsufficientFundsException
* InvalidPaymentException
* SameUserTransferException

---

## ▶️ Как запустить

1. Склонировать репозиторий:

```
git clone https://github.com/your-username/payment-system.git
```

2. Открыть в IntelliJ IDEA

3. Запустить `Main.java`

## 📌 Пример использования

```
1. Создать пользователя

3. Снять деньги
...
```

## 💡 Особенности

* Реализован rollback при ошибке перевода
* Используется Optional вместо null
* Чёткое разделение слоёв
